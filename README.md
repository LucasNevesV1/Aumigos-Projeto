# 🐾 AuMigos - Em Desenvolvimento

O **AuMigos** é uma plataforma web desenvolvida com o objetivo de auxiliar organizações e iniciativas de cuidado animal no gerenciamento de suas operações, promovendo mais organização, transparência e engajamento com a sociedade.

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)** e tem como foco central facilitar o controle de informações relacionadas a:

* 🐶 **Animais:** Cadastro, status e histórico.
* 💰 **Doações:** Controle de doações financeiras e materiais.
* 📦 **Estoque:** Gestão de insumos e mantimentos.
* 📊 **Financeiro:** Controle de entradas e saídas.

A plataforma busca reduzir a desorganização comum em ONGs e protetores independentes, além de incentivar a participação da comunidade na causa animal.

---

## 🚀 Funcionalidades

- [x] Cadastro e gerenciamento de animais.
- [x] Registro de doações (financeiras e materiais).
- [x] Controle de estoque de insumos.
- [x] Gestão financeira (entradas e saídas).
- [x] Visualização de relatórios gerenciais.
- [x] Sistema de cadastro e autenticação de usuários.
